ğŸš€ Step 3 â€” Configure the server with Easy Setup

Each server folder (1.TaoClassic, 2.JinClassic, 3.ZeroClassic) includes an Easy Setup tool to quickly prepare your configuration.

  1. Open Easy Setup from the server you want to run (e.g. C:\TripleServer\1.TaoClassic\EasySetup.exe).
  2. Server Name: Enter the display name you want (e.g. TaoClassic).
  3. Server IP:
    • Local â€” sets 127.0.0.1. Only this machine/VPS can connect (solo testing).
    • LAN â€” sets your private address (e.g. 192.168.x.x). Devices on the same router/network can join.
    • External â€” sets your public IP or DNS (for Internet play or VPS). Ensure required ports are open to the world.
  4. Server Port: Keep defaults unless needed.
  5. Database: Type the MySQL database name (e.g. taoclassic).
  6. Save: Click Save to write the configuration.
  7. Create client .dat: After saving, click Create Server.dat, Worlds.dat, and Oem.dat. Copy the generated files to your client folder (usually Client\ini\) and overwrite existing files.

Notes

  • For External play on a VPS, allow inbound Login Port,MSG Port and NPC Port in Windows Firewall.
  • If you change any port, re-create the .dat files and copy them to the client again.

Start the CrazyTao server first, then start Eudemons and Zero. This order helps avoid conflicts and ensures a smooth startup.
